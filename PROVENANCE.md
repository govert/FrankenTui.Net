# Provenance

## Upstream Basis

FrankenTui.Net is an intentionally traceable .NET port of
[FrankenTUI](https://github.com/Dicklesworthstone/frankentui), the Rust project
authored by Jeffrey Emanuel.

The project is not framed as a clean-room reimplementation of similar ideas. It
is framed as a port, translation, and adaptation of FrankenTUI into another
implementation medium while preserving provenance and upstream rights.

Initial upstream tracking point for this repository bootstrap:

- Upstream repository: `Dicklesworthstone/frankentui`
- Upstream branch policy: track `main`
- Bootstrap reference commit: `7a91089366bd4644e086d5a422cb76b052e3de17`
- Reference date: 2026-03-09

## Rights Allocation

Two categories of material may appear in this repository:

1. FrankenTUI-derived material
   Any code, documentation, tests, structure, or other expression copied from,
   translated from, ported from, adapted from, or otherwise derived from
   FrankenTUI.

2. Original FrankenTui.Net material
   Material authored specifically for FrankenTui.Net that is not derived from
   FrankenTUI expression.

FrankenTUI-derived material remains subject to the upstream FrankenTUI license,
including the OpenAI/Anthropic rider, as reproduced in
[LICENSES/FrankenTUI-UPSTREAM.txt](./LICENSES/FrankenTUI-UPSTREAM.txt).

Original FrankenTui.Net material is copyright 2026 FrankenTui.Net contributors
and is licensed under the MIT License reproduced in
[LICENSES/MIT.txt](./LICENSES/MIT.txt).

## Default Interpretation Rule

Because this repository exists to produce a faithful port, implementation work
should be presumed upstream-derived unless a file or note clearly states that it
is independent original material.

That default rule is intentional. It avoids accidental over-claiming of rights
in work whose purpose is to carry FrankenTUI expression into .NET rather than to
replace it with unrelated expression.

## No Waiver of Upstream Rights

Nothing in FrankenTui.Net is intended to narrow, waive, supersede, or obscure
any right retained by Jeffrey Emanuel or any restriction imposed by the upstream
FrankenTUI license rider.

Where a file, change, or deliverable contains both original FrankenTui.Net
material and FrankenTUI-derived material, treat the combined work as including
FrankenTUI-derived material and comply with the upstream FrankenTUI license for
that portion.
